# 🎬 Camera Fix - Quick Reference Card

## ⚡ Quick Fixes

### If You See "No Camera Found"

**STEP 1: Allow Permission**
```
Browser will ask → Click "Allow"
If no prompt → Check browser privacy settings
```

**STEP 2: Check Browser Settings**
```
Chrome: Settings → Privacy → Camera → Allow localhost
Firefox: about:preferences → Privacy → Camera → Allow
Edge: Settings → Privacy → Camera
Safari: Preferences → Websites → Camera
```

**STEP 3: Ensure HTTPS**
```
Development: OK with localhost:port
Production: MUST use HTTPS certificate
```

**STEP 4: Close Other Camera Apps**
```
Close: Zoom, Teams, Meet, Skype
Close: Other browser tabs using camera
Restart: Browser
```

**STEP 5: Try Different Browser**
```
Best: Chrome or Firefox
Also works: Edge
Limited: Safari
```

## 📱 Browser Permission Check

**Chrome/Edge:**
```
1. Click 🔒 in address bar
2. Find "Camera"
3. Set to "Allow"
4. Refresh page
```

**Firefox:**
```
1. Address bar → about:preferences
2. Privacy → Permissions
3. Camera: Find site → Change to Allow
```

## 🛠️ For Developers

**Check Available Cameras (Console):**
```javascript
navigator.mediaDevices.enumerateDevices()
  .then(devices => console.log(devices))
```

**Check Permission Status:**
```javascript
navigator.permissions.query({name: 'camera'})
  .then(result => console.log(result.state))
```

**Run App:**
```bash
flutter run -d web
```

**Check Logs:**
Open F12 → Console tab → Look for emojis

## 📋 Error Messages & Solutions

| Error | Solution |
|-------|----------|
| "Camera Not Available" | Allow permission in browser |
| "No Camera Found" | Check if camera in use / Try different browser |
| Permission Denied | Open browser settings → Allow camera |
| HTTPS Required | Use production HTTPS or localhost |
| Camera Stuck | Restart browser and device |
