🚀 QUICK START:
───────────────

FOR USERS GETTING "NO CAMERA FOUND":

1. Allow Camera Permission
   └─ Browser will ask when you first load the app
   └─ Click "Allow" button

2. Check Browser Settings
   ├─ Chrome: Settings → Privacy → Camera
   ├─ Firefox: Preferences → Privacy → Camera
   ├─ Edge: Settings → Privacy → Camera
   └─ Safari: Preferences → Websites → Camera

3. Ensure HTTPS or Localhost
   └─ Production: Must use HTTPS
   └─ Development: localhost:port is fine

4. Close Other Camera Apps
   └─ Zoom, Teams, Meet, etc.
   └─ Other browser tabs

5. Try Different Browser
   └─ Chrome/Chromium (recommended)
   └─ Firefox
   └─ Edge

═══════════════════════════════════════════════════════════

📊 TESTING CAMERA ACCESS:
──────────────────────────

1. Open Browser Developer Console (F12)
2. Run this command:
   navigator.mediaDevices.enumerateDevices()
3. Look for "kind: videoinput" in the output
4. If no videoinput, camera is not available
